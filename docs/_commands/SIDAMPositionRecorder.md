<p class="function_definition">SIDAMPositionRecorder(<span class="function_variables">grfName</span>)</p>

Show a panel to record positions.

## Parameters

**grfName :** ***string, default `WinName(0,1,1)`***  
The name of window.

## Returns
***variable***  
* 0: Normal exit
* !0 Error in input parameters
