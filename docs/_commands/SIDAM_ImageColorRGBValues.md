---
<p class="function_definition">SIDAM_ImageColorRGBValues(<span class="function_variables">grfName, imgName, key, RGBColor &s</span>)</p>

Returns values of minRGB/maxRGB.

## Parameters

**grfName :** ***string***  
The name of window

**imgName :** ***string***  
The name of an image.

**key :** ***string***  
"minRGB" or "maxRGB"

**s :** ***STRUCT RGBColor***  
rgb color is returned.

## Returns
***int***  
* 0: No error
* !0: Any error
